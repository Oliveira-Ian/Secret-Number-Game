# Secret Number Game 🎯

A simple interactive game where the player tries to guess a randomly generated number within a defined range.

This project was developed during my studies in JavaScript to practice fundamental concepts such as variables, functions, conditionals, DOM manipulation, and user interaction.

## 🚀 Features

* Random number generation
* Input validation
* Feedback messages to guide the player
* Attempt counter
* Option to restart the game
* Responsive layout

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript

## 🎮 How the Game Works

1. The system randomly generates a secret number.
2. The player must guess the number by typing a value in the input field.
3. The game informs if the secret number is **higher or lower** than the guess.
4. The player continues trying until the correct number is found.
5. After guessing correctly, the player can start a new game.
